使用fontcreator

1、从B字体拷贝字体到A字体的方法
	1、从B字体选择字体 复制
	2、在A字体中 Insert -> Glyphs -> 调整插入的个数 -> Before each selected glyph
	3、Edit -> Paste Special -> 勾选（Outline Data、Metrics、Glyph Name、Codepoints）-> Overrule same codepoints

2、头尾显示不全问题
	1、双击字符
	2、调整文字的位置
	3、顶部不能超过 WinAscent
	4、底部不能超过 WinDescent

3、复制字体到C:\Windows\Fonts
4、重启机器
